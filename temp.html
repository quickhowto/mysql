Copy a Table from Database A on Server A to Database B on Server B using HeidiSQL

HeidiSQL 9.4

  1. With Database A already created and present on Server A, launch HeidiSQL
  2. Connect to Server A (File > Session Manager)
  3. Connect to Server B (File > Session Manager)
  4. Both serves now in left pane
  5. In left pane
    1. Expand Server A (click +)
    2. Expand Database A
    3. Right-click on desired Table (that is to be copied)
    4. Select Export Database as SQL
  6. SQL export tab opens
    1. Only check to be on Table(s): Create
    2. Data: select Delete + Insert (truncate existing...
    3. Output: dropdown > select Server B
    4. Database dropdown populates below, select Database B
  7. Click Export
